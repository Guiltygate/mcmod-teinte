
block naming structure:

	UnlocalizedName -> only pertains to the lang file.
	I.e. UnlocalizedName -> en_us.lang
	Sets what name the player will see.
	
	
	RegistryName -> blockstate file name, which contains the -> block model name.
	I.e. RegistryName -> <SRC>.blockstates -> <SRC>.models.block
	Dictates how the block appears when placed.
	
	
	ModelResourceLocation -> models.item -> models.block
	Sets the block appearance as item, I.e. in hand, thrown on floor, in inventory.
	
	
	models.block -> textures.block
